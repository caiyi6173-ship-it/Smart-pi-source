# 中医养生四大基石

中医养生保健常围绕情志、饮食、起居、运动四个方面展开。它们属于低风险健康促进内容，适合在 RAG 中作为一般性建议，但不应替代疾病治疗。

情志调摄强调保持情绪相对稳定，避免长期过度紧张、焦虑、愤怒或抑郁。饮食调养强调规律、适量、清洁，并根据季节、体质、年龄和健康状态调整。起居调养强调作息规律、顺应昼夜节律。运动调养强调适度，不宜过量。

中医养生讲究因人而异。老人、儿童、孕产妇、慢病患者、术后恢复者、体质偏弱者，不应照搬普通人的运动、饮食或调理方法。

RAG 回答养生问题时，可以优先给出温和、通用、低风险建议，例如规律作息、适度运动、均衡饮食、保持情绪稳定、避免自行乱用药物。涉及药膳、方药、穴位强刺激、艾灸等内容，应加入适用边界和风险提示。
